🔍 Discover the Rule
❓ لاحظ الفرق بين الجمل دي:
🔹 I like — لما أتكلم عن حاجة بحبها عموماً
✅ "I like pizza" = أنا بحب البيتزا.
🔹 I'd like — لما أطلب حاجة دلوقتي (أدب أكتر)
✅ "I'd like a pizza" = عايز بيتزا (بأدب).
🔹 I don't like — لما أتكلم عن حاجة مش بحبها
✅ "I don't like spicy food" = مش بحب الأكل الحار.
🧠 Grammar Lab – like / don't like / would like
➕ like
I like chicken.
He likes rice.
➖ don't like
I don't like fish.
She doesn't like meat.
❓ Question
Do you like pasta?
Does he like salad?
🌟 would like
I would like a coffee.
What would you like? (طلب بأدب)
